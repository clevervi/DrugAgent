# Skill: Mejora la diversidad de los candidatos generados mediante la aplicación de técnicas de scaffold hopping avanzadas

### Descripción de la skill
La skill `diversity_enhancer_v3` se encarga de mejorar la diversidad de los candidatos generados mediante la aplicación de técnicas de scaffold hopping avanzadas. Esto se logra mediante la utilización de algoritmos de búsqueda más sofisticados y la incorporación de conocimientos químicos adicionales.

### Funcionamiento
La skill `diversity_enhancer_v3` se activa automáticamente después de cada iteración de generación de candidatos. Luego, se aplica a los candidatos generados para mejorar su diversidad y aumentar la probabilidad de encontrar nuevos compuestos con alta afinidad de unión.
