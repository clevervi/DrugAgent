# Skill: Mejora el filtro de calidad química (QED) para seleccionar candidatos con mayor probabilidad de tener alta afinidad de unión

### QED Filter Enhancer

Este filtro utiliza un conjunto de reglas y criterios para seleccionar candidatos con mayor probabilidad de tener alta afinidad de unión. Se basa en la calidad química (QED) y otros parámetros para evaluar la viabilidad de cada candidato.
