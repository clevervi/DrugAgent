# Skill: Aumenta la diversidad de las moléculas generadas mediante la introducción de nuevos grupos funcionales y la modificación de la estructura del esqueleto.

### Diversidad Enhancer

Este skill utiliza técnicas de química computacional para aumentar la diversidad de las moléculas generadas. Se pueden introducir nuevos grupos funcionales y modificar la estructura del esqueleto para crear moléculas más variadas.
