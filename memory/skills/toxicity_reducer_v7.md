### Toxicity Reducer v7

Esta skill utiliza un enfoque de aprendizaje automático para predecir la toxicidad de los candidatos y reducirla. Se entrenará con un conjunto de datos de candidatos conocidos y se utilizará para predecir la toxicidad de los nuevos candidatos generados.
