# Skill: Mejora los filtros de toxicidad para minimizar la generación de candidatos tóxicos

### Toxicity Filter Enhancer

Este skill utiliza algoritmos de aprendizaje automático para mejorar los filtros de toxicidad y minimizar la generación de candidatos tóxicos.

**Funcionalidad**
* Analiza los datos de toxicidad de los candidatos generados
* Identifica patrones y características que se asocian con la toxicidad
* Mejora los filtros de toxicidad para minimizar la generación de candidatos tóxicos
